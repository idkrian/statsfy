# Statsfy
Statsfy é um web app que trás dados sobre o seu perfil do spotify, como músicas e artistas mais escutados, estilos de música favoritos entre outros.

## Ferramentas usadas na criação
Foram utilizados ReactJS, Material UI, CSS e requisições API.

## Como Rodar:
Abrir o site https://statsfy.netlify.app/
<br />
ou
<br />
1 - Clonar ou baixar o repositório
<br />
2 - Utilizar "npm install" para instalar as dependências
<br />
3 - Utilizar o "npm start" para rodar o projeto
